# Guide du ChatBot EXED Manager 365

## Vue d'ensemble

Le chatbot est un assistant virtuel intelligent qui vous aide à naviguer et utiliser l'application EXED Manager 365. Il s'adapte automatiquement à votre rôle et à vos permissions pour vous fournir des réponses personnalisées.

## Accès au ChatBot

### Bouton flottant
- Situé en bas à droite de chaque page
- Icône de bulle de message (💬)
- Toujours accessible, peu importe où vous êtes dans l'application

### Comment l'utiliser
1. Cliquez sur le bouton flottant pour ouvrir le chat
2. Tapez votre question dans le champ de saisie
3. Appuyez sur Entrée ou cliquez sur le bouton d'envoi
4. L'assistant vous répond en quelques secondes

### Contrôles
- **Minimiser** : Cliquez sur l'icône `-` pour réduire la fenêtre
- **Fermer** : Cliquez sur l'icône `×` pour fermer complètement le chat
- **Rouvrir** : Cliquez à nouveau sur le bouton flottant (votre historique est conservé)

## Exemples de conversations

### Exemple 1 : Gestionnaire Scolarité
**Vous** : Comment créer un programme INTRA ?

**Assistant** : Pour créer un programme INTRA, suivez ces étapes :
1. Allez dans le module "Programmes"
2. Cliquez sur "Nouveau programme"
3. Remplissez le titre et le code
4. Sélectionnez "INTRA" comme type
5. Choisissez un client existant ou créez-en un nouveau
6. Définissez les dates si nécessaire
7. Cliquez sur "Créer"

Un programme INTRA est destiné à un client spécifique, contrairement aux programmes INTER qui sont ouverts à tous.

### Exemple 2 : Financier
**Vous** : Comment voir les factures impayées ?

**Assistant** : Pour voir les factures impayées :
1. Accédez au module "Factures"
2. Utilisez le filtre de statut en haut de la page
3. Sélectionnez "Envoyée" pour voir les factures non payées
4. Vous pouvez également voir "Partielle" pour les factures partiellement payées

Pour suivre les paiements, consultez le module "Recouvrements" où vous pouvez enregistrer les paiements et gérer les relances.

### Exemple 3 : Enseignant
**Vous** : Où puis-je voir mes prochains cours ?

**Assistant** : En tant qu'enseignant, vous pouvez :
1. Accéder à votre "Portail Enseignant" depuis le menu
2. Y voir tous les modules auxquels vous êtes affecté
3. Consulter les dates et horaires de vos interventions
4. Accéder aux documents de cours dans le module "Documents"

Vous pouvez également voir vos affectations dans le module "Classes" en cliquant sur une classe spécifique.

## Limitations

### Ce que le chatbot PEUT faire
✅ Expliquer comment utiliser chaque module
✅ Guider étape par étape pour les actions
✅ Expliquer les concepts (INTER/INTRA, statuts, etc.)
✅ Recommander les bonnes pratiques
✅ Vous orienter selon votre rôle

### Ce que le chatbot NE PEUT PAS faire
❌ Exécuter des actions directement (créer, modifier, supprimer)
❌ Accéder à vos données en temps réel
❌ Répondre à des questions hors sujet de l'application
❌ Modifier vos permissions ou votre rôle

## Sécurité et Confidentialité

- Toutes les communications passent par un serveur sécurisé
- Votre rôle est vérifié côté serveur
- Aucune donnée sensible n'est exposée
- Les conversations ne sont pas enregistrées
- L'historique se réinitialise quand vous fermez le chat

## Dépannage

### Le chatbot ne répond pas
1. Vérifiez votre connexion internet
2. Actualisez la page
3. Fermez et rouvrez le chat
4. Si le problème persiste, contactez l'administrateur

### Message d'erreur "Limite de requêtes"
- Attendez quelques secondes avant de réessayer
- Le système a une limite de requêtes par minute pour tous les utilisateurs

### Message d'erreur "Crédits épuisés"
- Contactez l'administrateur du système
- Les crédits AI doivent être rechargés

## Conseils d'utilisation

### Pour obtenir les meilleures réponses
1. **Soyez spécifique** : "Comment créer un programme INTRA ?" plutôt que "Comment créer ?"
2. **Une question à la fois** : Évitez les questions multiples
3. **Utilisez un langage naturel** : Pas besoin de mots-clés, parlez normalement
4. **Reformulez si nécessaire** : Si la réponse n'est pas claire, reformulez votre question
